# Baldur's Gate 1 Character Simulator

Python simulator that evaluates the probability of rolling enough points to create your **perfect Baldur's Gate 1 character**.

---

## Features

- Add, edit and load character builds from `power_builds.py`
- Select your build interactively from the list when running `main.py`
- Character generator (`character_generator.py`) uses **OOP and class inheritance** to compile character builds according to BG/AD&D rules
- Simulator (`simulator.py`) validates builds and rolls the dice to calculate probabilities

---

## 🚀 How to Run

1. Download the repository (Code → Download ZIP)
2. Extract the folder
3. Run: python main.py
